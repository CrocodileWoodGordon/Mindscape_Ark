# Mindscape Ark

Mindscape Ark 是一款俯视角叙事动作游戏。你将从 50 层宿舍区出发，穿越感官实验室、记忆档案馆与镜像空间，在方舟深处找回自身的答案。

## 下载方式（请先看这里）
在本页面右侧的 **Releases** 栏中选择与你的系统匹配的版本下载并运行。

## 当前可体验流程
- 章节链路：F50 宿舍区 → F40 感官实验室 → F35 记忆档案馆 → F30 逻辑核心 → F25 情感共鸣器 → F15 镜像空间 → F10 避难所 → F0 终章结算。
- 关键体验：启动故障序列 + 引导开场、战斗与解谜并行、分支选择与剧情回闪、最终结算与多种觉醒度结果。

## 楼层内容概览
- **F50 宿舍区**：开机序列与教学对白，探索相框与终端后解锁电梯。
- **F40 感官实验室**：抽象实验室地图、逻辑错误实体对话与清理任务、陷阱与能量墙分流。
- **F35 记忆档案馆**：迷宫式探索与记忆吞噬者战斗，多阶段 Boss 与回闪片段。
- **F30 逻辑核心**：服务器联动开关解谜，完成后解锁终端与霰弹枪缓存。
- **F25 情感共鸣器**：六位情绪回声对话与速度增益，击败中央异常后解锁通行。
- **F15 镜像空间**：中轴线锁定机制、镜像同步战斗与镜像 Boss，掉落熵能步枪。
- **F10 避难所**：与艾拉互动触发五波防御战或使用认知溶解剂的分支选择。
- **F0 终章**：系统结论与觉醒度结算，回到标题界面。

## 核心玩法系统
- **移动与寻路**：WASD 自由移动 + 右键 A* 自动寻路，路径阻塞自动重规划。
- **战斗循环**：指向射击、装填管理、受击闪光、敌人追击与近战判定。
- **武器切换**：手枪 / 霰弹枪 / 熵能步枪，按 `1-3` 切换已解锁武器。
- **交互与任务**：F 键互动、阶段任务提示、终端/开关/电梯推进。
- **存档与成就**：暂停菜单可暂存进度，标题页读取存档；成就弹窗与成就列表可查看。
- **界面反馈**：任务提示、最小地图、生命/弹药/装填 HUD。

## 操作方式
- `WASD` / Arrow keys — 手动移动（支持对角线；相反方向会互锁直到松开）。
- Mouse right-click — A* 自动寻路，无法直达时退化到最近可达格。
- Mouse left-click / `Space` — 朝鼠标方向射击（步枪支持按住连发）。
- `F` — 触发提示的互动（终端、相框、开关、电梯、NPC）。
- `R` — 装填弹匣。
- `1`/`2`/`3` — 切换武器栏位（需先解锁）。
- `Esc` — 暂停菜单（终章为返回标题）。
- `Enter` — 对话确认、死亡后返回标题。

## 存档与成就
- 暂停菜单选择“暂存进度”写入存档，可生成多条记录（按时间排序）。
- 标题页“读取存档”可按时间与楼层恢复进度。
- 成就可在暂停菜单打开，页面内显示完成数量与列表进度。
